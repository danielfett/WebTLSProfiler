Web TLS Profiler

The Web TLS Profiler is a web interface for the TLS Profiler Python package. The TLS Profiler scans the TLS settings of a webserver and compares the configuration with a Mozilla TLS Profile.
Mozilla provides three TLS profiles for a different security need and client support. The Old Profile has the best support for legacy clients, the Intermediate Profile should be used for the most endpoints and the Modern Profile provides the highest level of security. You can find a detailed description of the profiles on the Mozilla wiki page Security/Server Side TLS.
Just enter your domain address in the form field and choose a profile from the dropdown menu.
